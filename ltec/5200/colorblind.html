Description

  • 8 percent of males 0.5 percent of women of European ancestry have a color blindness ("Facts About Color Blindness", 2015)
  • While usually hereditary, color blindness is can be caused by trauma, UV damage or shaken baby syndrome ("Color Blindness Causes")

Different Kinds of Color Blindness

Anomalous Trichromat vision- all color receptors are used, but one pigment is misaligned: (Swierenga, 2002)

  • Protanomaly- reduced sensitivity to red (Swierenga, 2002)
  • Deuteranomaly- reduced sensitivity to green, most common (Swierenga, 2002)
  • Tritanomaly- reduced sensitivity to blue

Dichromat vision- where only 2 of the 3 color pigments are used

  • Protanopia- unable to see red (Swierenga, 2002)
  • Deuteranopia- unable to see green (Swierenga, 2002)
  • Tritanopia- unable to see blue (Swierenga, 2002)

Monochromat (Achromatopsia) vision individuals that can only see one color:

  • Typical/Rod Monochromatic - unable to combine colors, fully grayscale (Swierenga, 2002)
  • Atypical/Cone Monochromatic - very low color recognition. (Swierenga, 2002)

Challenges

  • Certain color combinations makes it difficult for colorblind to use GUI’s and touch screens
  • Images such as charts or maps that use color to represent data are examples of some colorblind challenges (José de Araújo, Cesar Dos, & Bonacin, 2018)

Developer Tips

  • The WCAG guidelines recommend a contrast ratio of at least 4.5:1 for text and its background. Try using this color contrast analyser tool.
  • One can view their website in the way a color blind person would see it using this color filter
  • Ideally, design a site in black and white and only add color for emphasis (Swierenga, 2002)
  • Color combinations that should be avoided are: red-green, green-yellow, and blue-yellow. Also, make sure that the colors have sufficient contrast to increase readability. (Swierenga, 2002)
  • If color coding is used on a website, be sure to add additional visual indications like an icons to help differentiate parts of the site.
