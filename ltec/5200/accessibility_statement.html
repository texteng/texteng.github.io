Accessibility Statement for Web Accessibility For All

Stephen Teng is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone, and applying the relevant accessibility standards.

Measures to support accessibility

Stephen Teng takes the following measures to ensure accessibility of Web Accessibility For All:

  • Include accessibility as part of our mission statement.

Conformance status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. Web Accessibility For All is not assessed with WCAG 2.1 level AA. Not assessed means that the content has not been evaluated or the evaluation results are not available.

Additional accessibility considerations

This is a page is a graduate school project which did not have accessibility as a requirement. The author implemented the accessibility requirements to the best of his ability.
